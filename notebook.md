## Básicos

**Transpilar**:

```bash
tsc nombre_archivo.ts
```

**Inicializar (tsconfig.json)**

```bash
tsc --init
```

Tras la creación de este archivo, para transpilar todos los archivos `.ts` con: `tsc` es suficiente.

**Watch Mode**

```bash
tsc --watch
```

```bash
tsc -w
```
